typedef Error(* HAL::REST::RestServer::URIHandler) (char *buffer, size_t length)

Convenience definitions for URI handlers called when an endpoint is triggered.

The input buffer, along with the error code is used to populate the server response. An error of HTTP_200 sends either the GET request payload or 'successful' POST request message. Non-HTTP_200 error codes send any error message contained in the buffer when the handler returns.

Parameters
bufferThe buffer containing JSON payload.
lengthThe length of the payload.
Returns
Error HTTP status for the request.
